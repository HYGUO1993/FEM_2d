# ⚠️ 如果 GUI 无法启动 - 快速修复

## 问题：提示"安装依赖失败"

### 🔧 快速修复（3 步）

#### 第 1 步：双击依赖安装脚本

在项目根目录找到并双击：
```
安装依赖.bat
```

这会尝试安装 PySide6、matplotlib 和 numpy。

---

#### 第 2 步：如果第 1 步失败，手动安装

打开命令行（PowerShell 或 CMD），运行以下命令之一：

**选项 A（最可能成功）：**
```bash
python -m pip install --user PySide6 matplotlib numpy
```

**选项 B（如果使用 Anaconda）：**
```bash
conda install -c conda-forge PySide6 matplotlib numpy
```

**选项 C（一个包一个包）：**
```bash
python -m pip install --user PySide6
python -m pip install --user matplotlib
python -m pip install --user numpy
```

---

#### 第 3 步：验证安装成功

```bash
python -c "import PySide6, matplotlib, numpy; print('SUCCESS!')"
```

如果看到 `SUCCESS!`，表示安装成功。

---

## 现在启动 GUI

安装完成后，再次尝试：

1. **双击启动**：`启动GUI.bat` 或 `start_gui.bat`
2. **或命令行**：`python quick_start_gui.py`
3. **或 VS Code**：`Ctrl+Shift+P` → 搜索 `Launch FEM_2d GUI`
