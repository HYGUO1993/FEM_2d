## 快速开始

### Windows 用户
#### 方式 1：双击批处理文件（推荐）
1. 打开文件资源管理器
2. 导航到项目根目录 → `scripts` 文件夹
3. 双击 `run_gui.bat` 启动应用

#### 方式 2：PowerShell
```powershell
# 在项目根目录打开 PowerShell，运行：
.\scripts\run_gui.ps1
```

### macOS / Linux 用户
```bash
# 在项目根目录执行：
python3 python/fem_gui.py
```

## 环境要求

### 系统要求
- **操作系统**：Windows 7+、macOS 10.13+、Linux（任何现代发行版）
- **Python**：3.8 或更高版本
- **RAM**：至少 512 MB

### 已编译的 FEM 求解器（可选但推荐）
- C++ Release 版本的 `fem_run.exe`（或 Linux/macOS 下的 `fem_run`）
- 用于实际的有限元求解计算

## 安装依赖

### 自动安装（推荐）
启动脚本会自动检测并安装缺失的 Python 包：

```bash
# 通过 run_gui.bat 或 run_gui.ps1 启动时会自动安装
```

### 手动安装
如果自动安装失败，手动安装：

```bash
# 使用 pip
pip install PySide6 matplotlib numpy

# 或使用 conda
conda install -c conda-forge PySide6 matplotlib numpy
```

应用启动后，您应该看到：

## 基本操作

### 菜单说明

#### 文件菜单 (File)
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| New | Ctrl+N | 创建新模型（清空当前内容） |
| Open | Ctrl+O | 打开已有的模型文件（.txt/.json） |
| Open Example | — | 打开项目示例文件 `test05.txt` |
| Save | Ctrl+S | 保存当前模型 |
| Save As | Ctrl+Shift+S | 使用新名称保存模型 |
| Exit | Ctrl+Q | 关闭应用（若有未保存内容会提示） |

#### 编辑菜单 (Edit)
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| Undo | Ctrl+Z | 撤销（暂未实现） |
| Redo | Ctrl+Y | 重做（暂未实现） |

#### 分析菜单 (Analysis)
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| Solve | F5 | 运行求解器并生成可视化结果 |
| Export Plot | — | 将当前图像导出为 PNG/JPEG/SVG 文件 |

#### 查看菜单 (View)
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| Show Deformed | F6 | 显示/隐藏变形图 |
| Show Forces | F7 | 显示/隐藏按轴向力着色的杆 |
| Show Reactions | F8 | 显示/隐藏支座反力箭头 |

#### 帮助菜单 (Help)
| 功能 | 快捷键 | 说明 |
|------|--------|------|
| About | — | 显示应用版本和功能介绍 |

### 工具栏快捷键
工具栏提供常用操作的快速访问：
- 📄 新建、打开、保存按钮
- ▶️ 求解按钮（绿色播放图标）
- 💾 导出图像按钮

### 可视化控制面板
右侧边栏"Visualization Controls"提供交互式调整选项：

| 控制项 | 默认值 | 用途 |
|-------|--------|------|
| Deformation scale | 1000.0 | 调整变形倍数（数值越大，变形越明显） |
| Reaction scale | 0.05 | 调整支座反力箭头的显示尺度 |
| Show deformed shape | ✓ | 显示/隐藏变形后的结构 |
| Color by axial force | ✓ | 根据轴向力对杆着色（红=压力，蓝=拉力） |
| Show reactions | ✓ | 显示/隐藏支座反力箭头 |
| Redraw | — | 手动刷新可视化图像 |

### 中央可视化区域
- **图像显示**：Matplotlib 交互式画布
  - 📍 **平移**：点击并拖动鼠标
  - 🔍 **缩放**：滚动鼠标滚轮或使用工具栏缩放按钮
  - 🏠 **复位**：点击工具栏"Home"按钮恢复原始视图
  - 💾 **保存**：点击工具栏"Save"按钮保存截图
- **运行日志**：下方面板显示所有操作日志和求解过程信息

## 常见任务

### 任务 1：加载示例模型并查看
1. 启动应用
2. 点击 **File** → **Open Example (test05)** 或按快捷键 **F5** 直接求解
3. 点击 **Analysis** → **Solve**（或按 **F5**）
4. 等待求解完成，中央区域会显示变形图和支座反力

**预期结果**：
- 显示一个梁单元的变形图
- 支座处有反力箭头
- 日志面板显示求解信息

### 任务 2：打开自己的模型
1. 准备一个 FEM_2d 格式的输入文件（.txt 格式）
2. 点击 **File** → **Open...** 或按 **Ctrl+O**
3. 选择您的模型文件
4. 点击 **Analysis** → **Solve**（或按 **F5**）
5. 查看结果图像

**输入文件格式**：
```
2 1 1 1 1 1
1 0 0
2 10 0
1 1 -1 -1
1 200000 0.3 0
1 0.01 8.33e-5 0.1
1 1 -1000 -1 1 0 0 0
```

### 任务 3：调整可视化参数
1. 在求解后，右侧面板"Visualization Controls"中调整各项
2. **Deformation scale**：增大数值可更明显地显示变形
3. **Reaction scale**：调整支座反力箭头的显示大小
4. 点击 **Redraw** 或直接在数值框修改后自动刷新

### 任务 4：导出结果
1. 求解完成后，点击 **Analysis** → **Export Plot...**
2. 选择导出格式（PNG/JPEG/SVG）和保存位置
3. 点击"保存"完成导出

## 故障排除

### 问题 1：启动时出现"Python not found"错误
**原因**：Python 未安装或不在 PATH 环境变量中

**解决方案**：
1. 确保已安装 Python 3.8 或更高版本
2. 如使用 Anaconda，激活相应环境后再启动脚本
3. 在 Windows 中，确保 Python 安装时选择了"Add Python to PATH"

### 问题 2：缺少依赖包（PySide6、matplotlib、numpy）
**原因**：Python 包未安装

**解决方案**：
```bash
pip install PySide6 matplotlib numpy
```

### 问题 3：应用启动但无法求解
**原因**：Release 版本的 `fem_run.exe` 未编译

**解决方案**：
```bash
# 在项目根目录执行
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

### 问题 4：图像显示不完整或格式错误
**原因**：结果文件解析错误或格式不符

**解决方案**：
1. 检查模型文件格式是否正确
2. 在日志面板查看详细错误信息
3. 尝试使用示例文件 `test05.txt` 验证

### 问题 5：点击"Solve"后没有反应
**原因**：
- 未打开模型文件
- 求解过程较长（大模型）

**解决方案**：
1. 确保已通过 **File** → **Open** 或 **Open Example** 打开模型
2. 观察日志面板，查看是否有进度信息
3. 对于大型模型，可能需要等待较长时间

### 问题 6：在 macOS/Linux 上无法启动
**原因**：脚本执行权限问题

**解决方案**：
```bash
# 赋予脚本执行权限
chmod +x scripts/run_gui.ps1

# 然后运行
python3 python/fem_gui.py
```

## 高级用法

### 环境变量
支持以下环境变量自定义行为（可选）：

| 变量 | 说明 |
|------|------|
| `FEM_OUTPUT_DIR` | 自定义结果输出目录（默认：`build/`） |
| `FEM_SOLVER_PATH` | 自定义求解器可执行文件路径 |

### 日志输出
所有操作和错误都记录在中央区域的日志面板中。日志前缀说明：
- `[INFO]` - 信息消息
- `[WARNING]` - 警告消息
- `[ERROR]` - 错误消息

### 结果文件位置
- **解析结果**：`build/Results_gui.dat`
- **生成图像**：`build/plot_gui.png`
- **导出图像**：用户指定的位置
